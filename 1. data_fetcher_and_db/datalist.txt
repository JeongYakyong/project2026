# 요약 / 특성 메모
################################################################################
 · DB 2종(jeju/land) 모두 historical / forecast 의 동일 골격.
     - historical : 2020-01-01 시작, 시간단위, 실측(real_*) 중심.
     - forecast   : 2025-12월~ 시작, 시간단위, 예측(est_*)/예보 중심.
 · 컬럼 네이밍 = "지표 + 지역접미사".
     - jeju:  _west / _east / _south
     - land:  _daegwallyeong / _wonju / _seosan / _pohang / _yeonggwang
 · 발전원 키워드: solar(태양광), wind(풍력), gas/LNG, oil(유류),
   coal(석탄), nuclear(원자력), hydro(수력), pumped(양수).
 · 가격 키워드: smp (_da 전일 / _rt 실시간), neg(음SMP).
 · CSV 해상도: monthly(gas_*) / daily(lng,oil) / hourly(hvdc,only_gen).
 · data/ 내 input_data_jeju.db.bak_20260603 은 백업본(목록에서 제외).
