## 0. 배경 (왜 조사하는가)

제주 전력시장 데이터로 **net_load → 발전용 LNG 수요**의 관계를 분석하려고 한다.
제주는 발전량을 연료원별로 직접 제공하지 않으므로, 아래 **도출식**으로 LNG(내연) 발전량을 역산할 계획이다.

```
LNG(내연) 발전량 ≈ 전체수요(전력거래량) − HVDC 수전전력 − 신재생 발전량(태양광+풍력)
```

이 조사의 **최종 목적은 단 하나** — 위 도출식이 실제로 계산 가능한지, 가능하다면
어떤 시간 해상도·기간·단위로 가능한지를 확정하는 것이다. 모든 검사는 이 질문에 수렴해야 한다.

추가로, 도출한 제주 LNG 수요를 KOGAS의 **발전용 가스 데이터(월별 기온효과 / 일별 공급량)** 및
**요금 단가**와 연결할 수 있는지(기간·해상도 정합)도 함께 본다.

## 1. 공통 작업 규칙 (모든 파일에 적용)

1. **인코딩**: 한국 공공데이터 CSV는 `utf-8`이 아닌 `cp949`/`euc-kr`인 경우가 많다.
   `utf-8` 실패 시 `cp949` → `euc-kr` 순으로 재시도하고, **최종 성공한 인코딩을 기록**할 것.
   (권장: `chardet`로 1차 감지 후 검증)
2. **헤더**: 상단에 메타 설명행·빈 행·병합셀이 있을 수 있다.
   실제 컬럼 헤더가 몇 번째 행인지, 헤더가 2줄 이상인지(다중 헤더) 확인.
3. **xlsx**: 시트가 여러 개일 수 있다. **시트명 전체를 나열**하고, 데이터가 든 시트를 식별할 것.
4. **단위 추론**: 컬럼명·값 크기로 단위를 추정하고 근거를 적어라.
   특히 발전량/수전전력이 **MW(순시값)인지 MWh(전력량)인지**가 뺄셈 정합에 치명적이다 — 반드시 판별.
5. 각 파일마다 **샘플 3행**(헤더 직후)을 원문 그대로 첨부.
6. 날짜/시간 컬럼은 **최소값·최대값·행 수**를 보고하고, 결측·중복 타임스탬프 여부를 적어라.

## 2. 파일 그룹과 그룹별 중점 확인 사항

### 그룹 A — 제주 시계열 (LNG 도출식의 핵심)
- `제주_시간대별발전량(16-24).csv`
  - **★최우선 확인★: 연료원별 분해 컬럼이 있는가?** (예: 태양광/풍력/내연/기력 등)
    → 있으면 신재생/내연을 직접 떼어낼 수 있어 도출식이 성립. 단일 총합 컬럼뿐이면 도출 불가.
  - 컬럼 전체 목록, 시간 해상도(시간별 맞는지), 단위(MW/MWh).
- `제주_시간대별전력거래량(17-25).csv`
  - 이 값이 **"전체수요"** 역할을 할 수 있는지(거래량=수요 proxy). 컬럼·단위·해상도.
- `제주_시간대별HVDC(17-25).csv`
  - HVDC 총 수전전력으로 쓸 수 있는지. **양방향(음수=역송) 값이 있는지**, 부호 규약 확인.
- `제주 일별 시간대별 연계선 수전전력(HVDC#1/#2/#3)_{250428,251103,260407}.csv` (총 9개)
  - 파일명 끝 숫자(250428=2025-04-28 등)는 **추출 시점 스냅샷**으로 추정됨.
  - **세 스냅샷이 같은 회선을 기간 연장한 것인지, 겹치는지** 확인 → 중복 제거/이어붙이기 전략 제시.
  - #1·#2·#3은 물리적으로 다른 회선으로 추정. **세 회선의 합 ≈ `제주_시간대별HVDC(17-25)`인지**
    겹치는 임의 시점 한 개를 골라 대조(둘 중 무엇을 쓸지 결정 근거가 됨).

### 그룹 B — 발전용 LNG / 가스 (KOGAS = 산자부 데이터, 자격요건 핵심)
- `daliy_lng_gen_21-26.csv` (※파일명 오타 daily)
  - **발전량(MWh)인가 가스 공급량(체적/열량)인가?** 전국 단위인가 지역 분해 가능한가? 해상도(일별 추정).
- `montly_lng_gen_temp.csv`
  - 월별 LNG 발전량 + 기온. **기존 "기온 민감도" baseline의 원천**으로 보임. 컬럼·기간 확인.
- `한국가스공사_총발전량 월별 기온효과_20210630.csv`
  - 위 `montly_lng_gen_temp`와 **같은 데이터인지 다른 데이터인지** 비교.
- `한국가스공사_한국의 월별 주요국산 천연가스 수입단가_20260131.csv`
  - 월별·국가별 수입단가. 컬럼 구조, 단위.

### 그룹 C — 요금 단가 (KOGAS, 산자부 데이터)
- `2020~2026년 발전용 천연가스 요금 단가.xlsx` (7개)
  - 해상도(월별 추정), 컬럼 구조(원료비/공급비 등 단가 항목), **단위(원/MJ·원/Nm³ 등)**.
  - **7개 파일의 컬럼 구조가 연도별로 동일한지** → 하나의 2020–2026 시계열로 합칠 수 있는지 판정.

### 그룹 D — 보조/연계 데이터
- `oil_price_20-26.csv` — 유가(LNG 가격 연동 변수 후보). 컬럼·해상도·단위·통화.
- `한국전력거래소_일별 화력발전량_20220630.csv` — KPX 전국 일별 화력발전. 컬럼·기간·단위.

## 3. 반드시 교차 검증할 항목 (파일 간)

1. **도출식 기간 교집합**: 발전량(16–24)·전력거래량(17–25)·HVDC(17–25) 세 파일의
   공통 가용 기간을 계산해 명시(도출 가능한 실제 윈도우).
2. **해상도 사다리**: 제주(시간별) → KOGAS 가스(일별/월별)로 올라가는 집계가 필요하다.
   각 가스 데이터의 정확한 해상도를 보고해, 어디서 일/월 집계가 필요한지 표로 정리.
3. **단위 정합**: 도출식 좌변 세 항(전력거래량·HVDC·발전량)의 단위가 뺄셈 가능하게 일치하는지.
4. **HVDC 중복 위험**: 그룹 A의 HVDC 총합 파일 vs 회선별 스냅샷 9개 — 어느 쪽을 단일 출처로
   쓸지 권고(중복 합산 방지).
