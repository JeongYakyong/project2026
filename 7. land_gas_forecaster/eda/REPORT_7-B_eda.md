> 목적: 제주 모델 착수 전 **명제(§1.2 검증목표 1: 신재생 → 잔여부하 → 가스)가 제주 데이터에 실제로 있는지** 확인.

## 핵심 결론 (한 줄)
제주는 **net_load↔가스 관계가 강하고(r=0.723) 안정적**이며, 전국에선 약했던 **신재생→가스 대체효과가 계량 신재생만으로 직접 뚜렷**(수요통제 신재생계수 **−0.369**)하다. **G-9 통과**, 데이터 결손 없음. 2024-06 시범사업 레짐은 구조적 단절을 만들지 않았다.

- 0 비중 **1.3%** — 제주 LNG는 거의 항상 켜져 있다(기저+첨두 혼합). 영 과다(zero-inflated) 아님 → 분류 임계 거동보다 회귀가 주효.
- 전체 평균 169.8MW, 최대 486.6MW.

## 2. net_load ↔ lng_gen 관계 (명제 그 자체)

- 구간 평균선이 우상향하며 **부하수준별 비선형**(저부하 평탄 → 고부하 급증, merit-order). → 트리/구간 모델 유리.
- 산출: `fig/7b_02_netload_vs_gas.png`, `tab/7b_02_netload_vs_gas_binned.csv`.

## 5. ★ 신재생 → 가스 대체효과 (명제의 심장)
| 측정 | 값 |
|---|---|
| 원상관 corr(신재생, 가스) | **−0.415** |
| 수요통제 부분상관(수요 10분위 내) 평균 | **−0.525** |
| 회귀 신재생계수(가스 ~ 수요 + 신재생) | **−0.369** → 신재생 +100MW 시 가스 **−37MW** |
| 태양광계수 / 풍력계수(분리 회귀) | **−0.368 / −0.368** (동등) |

- **수요를 통제하면 음(−) 관계가 오히려 더 또렷**(−0.415 → −0.525). 신재생이 가스를 직접 밀어낸다.
- 태양광·풍력 기여가 거의 동일 → 제주는 두 자원 모두 가스 대체에 기여.
- **★ 전국 대비**: 전국(7-0 5-b)은 침투율 낮고 자가소비 태양광이 수요에 숨어 신재생계수 ≈0 → 역추정(7-0b)으로 겨우 −0.33을 복원했다. 제주는 **계량 신재생만으로 −0.369가 바로** 나온다. §1.2 설계("신재생 많은 제주에서 입증")가 데이터로 확인됨.
- 산출 `fig/7b_05_renew_displacement.png`, `tab/7b_05_displacement_reg.csv`.

## 6. ★ 2024-06 제주 전력거래 시범사업 레짐 점검
- post(2024-06~12) **5,136행**. 전/후 net_load→가스 곡선 겹침구간 평균 |가스차| **27.8MW**.
- 전/후 corr **0.702 → 0.832** (유지·강화). 곡선 차이는 주로 **2024년 수준 상승(여름 중심 6개월) 혼입**이지 시장 기제 변화가 아님.
- 판정: **급전순위 전/후 동일 → 구조적 단절 없음.** 2020–2024 통합 학습 안전. 안전장치로 regime 라벨만 보존(모델 단계에서 사용 여부 선택).
- 배경(메모): 제주 시범사업으로 **상당량 PPA 용량이 Grid 계량에 흡수** → 계량 신재생만으로 충분, 별도 PPA 산정 불필요. (육지는 미흡수 → 육지 PPA 산정 포함이 필수. `jeju_ppa_cumulative_gen.csv`는 7-B에서 미사용, 이력 보존.)
- 산출 `fig/7b_06_regime_pilot.png`, `tab/7b_06_regime.csv`.
